# Mode: Full (AI + Bug)

Run the AI-readiness workflow first (Steps 1-3 of `references/mode-ai.md`), then the bug-assessment workflow (Steps 1-3 of `references/mode-bug.md`).

After both reports print, append the combined-verdict template from `references/report-templates.md` (`Combined Health Verdict` section).

The verdict should:

- State whether the codebase is ready for AI-assisted development
- Recommend what to fix first — structure or bugs
- Give a recommended order of operations
